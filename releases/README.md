# Releases

This folder holds installable test artifacts for people who want to try the app without building it first.

- `HeatMapV1Android-v2.1.apk` — Android release APK
- `HeatMapV1Android-v2.1.sha256.txt` — checksum for integrity verification

If you are evaluating the app on an Autel-style controller tablet, start with this APK before trying a local build.
